# こんとき — SST(ソーシャルスキルトレーニング)アプリ

「こんなとき、どうする?」を毎日1つ・3分だけ練習するアプリの企画・データ一式。
このディレクトリは仮置きで、iOS実装を始める際に専用リポジトリへ移す想定。

## 構成

- `scenarios/kontoki_data.json` — シナリオデータ本体(スキル「かして」週5場面、選択肢A/B/C・結果・きもちクイズ)。将棋タイトルと同じ「起動時にJSONを読む」構成を想定し、`dataVersion` で差し替え管理する
- `docs/scenario-design.md` — シナリオ設計書(スキル10種マップ、脚本、画像生成プロンプトと量産ルール)
- `docs/design-proposal.html` — デザイン案(画面モック・デザイン原則・コンテンツ編成・カラーパレット)
- `docs/prototype.html` — ブラウザで動く体験プロトタイプ(シナリオ「かして」1本収録)。ダブルクリックで開くだけで動く
- `ios/` — SwiftUI実装(未着手。ローカルのClaude Code + Xcodeで実装予定)

## データ仕様メモ

- 各シーンの選択肢は3系統固定:A=ことばで伝える(目標行動)/B=ことばを使わない行動/C=消極・回避
- どの選択肢にも罰を与えない。`stamp: true` はA系統のみ
- `quiz.answer` は `happy` / `sad` / `angry` / `surprised` の4値
- 実施ログは端末内保存・実名なし(ニックネームのみ)の方針
