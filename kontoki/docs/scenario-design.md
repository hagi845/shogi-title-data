# 「こんとき」シナリオ設計書 v1

アプリの本体となるコンテンツの生産ライン。**スキル10種×場面5つ=50場面**を供給単位とする。
登場人物は全シナリオ共通で **ゆうくん(主人公・青い服)/みおちゃん(相手役・おだんご頭・紫の服)/こんちゃん(マスコット)** に固定する(「知っている顔ぶれ」の安心感を般化の足場にする)。

## 1. スキル10種マップ(カリキュラム順)

| # | スキル | ねらい | 場面バリエーション(週の5日分) |
|---|--------|--------|--------------------------------|
| 1 | **かして** | ことばでお願いする | のり(教室)/ブランコ(公園)/ゲーム機(放デイ)/図鑑(図書室)/色えんぴつ(工作) |
| 2 | **じゅんばん** | 待つ・ゆずる | すべり台/水道/タブレット/レジごっこ/エレベーターのボタン |

| 4 | **てつだって** | 助けを求める | ジッパーが閉まらない/漢字がわからない/牛乳をこぼした/high い棚の物/道具が見つからない |
| 5 | **いやだよ** | 断る・嫌と伝える | 苦手な食べ物をすすめられる/遊びをかえたい/くすぐられる/物を貸したくない/帽子をとられそう |

**編成ルール**
- 1〜3が「入る・借りる」系、4〜7が「伝える」系、8〜10が「気持ち」系。この順で難度が上がる
- 週テーマ制:1週間同じスキル、場面だけ日替わり(般化)
- 選択肢は毎回3系統で固定:**A=ことばで伝える(目標行動)/B=ことばを使わない行動/C=消極・回避**
- どの選択肢にも罰を与えない。Bは「相手がどう感じたか」を見せ、Cは「困りごとが残ること」を見せて、Aへ誘導する

## 2. 「かして」週 完全脚本(5日分)

きもちクイズの選択肢は常に4つ:**うれしい/かなしい/おこってる/びっくり**

### 月曜:のり(教室)※プロトタイプ収録済み

- **場面**:みおちゃんが つかっている のりを、つかいたいな。どうする?
- **A「かして」って いってみる** → みおちゃん「いいよ!」とにっこり。✦「かして」が いえたね!ことばで つたえると、きもちが つたわるね
- **B だまって つかう** → みおちゃんは びっくりして かなしそう。「きゅうに のりが なくなって、かなしい きもちに なったんだって」→ こんちゃん「ことばで つたえると、きもちが つたわるよ」
- **C おわるまで まってみる** → しずかに まてたね。でも こうさくの じかんが おわっちゃうかも。「かして」って いってみたら どうなるかな?
- **きもちクイズ**:(Aのあとの みおちゃんの かお)→ こたえ:うれしい。ヒント「くちの かたちを みてみよう」

### 火曜:ブランコ(公園)

- **場面**:こうえんで ブランコに のりたいな。でも みおちゃんが のっている。どうする?
- **A「かわって」って いってみる** → みおちゃん「あと 10かい こいだらね!」。すこし まったら かわってくれた。✦じぶんの きもちも つたえて、あいての つごうも きけたね。りょうほう まるだよ
- **B ブランコの まえに たって じっと みる** → みおちゃんは こわくなって ブランコを やめちゃった。たのしい きもちが なくなっちゃったみたい。→ こんちゃん「ことばに すると、あいては あんしんするよ」
- **C あきらめて かえろうとする** → ブランコに のりたかった きもちは どこへ いくのかな。むねが もやもやしたまま だね。「かわって」って いってみたら?
- **きもちクイズ**:(Bのときの みおちゃんの かお)→ こたえ:びっくり。ヒント「めの おおきさを みてみよう」
- **設計メモ**:Aが「すぐ貸してもらえない(でも交渉成立)」パターン。即成功だけでなく「待つ約束」も成功体験にする

### 水曜:ゲーム機(放デイのじゆうじかん)

- **場面**:みおちゃんが ゲームを している。ぼくも やりたいな。どうする?
- **A「つぎ、やらせて」って いってみる** → みおちゃん「この めんが おわったらね」。やくそくどおり かわってくれた。✦「つぎ」って いえると、あいては じゅんびが できるんだね
- **B コントローラーを ひっぱる** → ゲームが とちゅうで きえちゃった。みおちゃんは おこった かお。「たいせつな データが きえて、いやな きもちに なったんだって」→ こんちゃん「あいてにも たいせつな ものが あるんだね」
- **C うしろで ずっと みている** → みることも たのしいね。でも 「やりたい」は つたわっていないみたい。こえに だして みようか
- **きもちクイズ**:(Bのときの みおちゃんの かお)→ こたえ:おこってる。ヒント「まゆげの かたちを みてみよう」

### 木曜:図鑑(としょしつ)

- **場面**:としょしつで よみたかった どうぶつずかんを、みおちゃんが よんでいる。どうする?
- **A「おわったら かして」って いってみる**(こえは ちいさめで) → みおちゃん「いっしょに みる?」。ならんで ずかんを みた。✦おねがいしたら、もっと いいことが おきたね
- **B よこから のぞきこんで ページを めくる** → みおちゃんは びっくりして ページが やぶれそうに。ひやっとしたね。→ こんちゃん「ほんは としょしつの みんなの ものだから、そーっとね」
- **C ちがう ほんを よんで まつ** → それも いい ほうほうだね!でも よみたい きもちが のこってるなら、「おわったら かして」って よやくも できるよ
- **きもちクイズ**:(Aのあとの みおちゃんの かお)→ こたえ:うれしい。ヒント「ほっぺを みてみよう」
- **設計メモ**:Cを部分的に肯定する回。回避が常に悪ではなく「+ことば」でもっと良くなる、を教える。図書室なので声の大きさの伏線(スキル9)も仕込む

### 金曜:色えんぴつ(こうさくの じかん)

- **場面**:みどりの 色えんぴつが つかいたい。みおちゃんが いま つかっている。どうする?
- **A「みどり、つかいおわったら かして」って いってみる** → みおちゃん「いいよ、はい!」。✦こんしゅう 5かいめの「かして」!もう めいじんだね
- **B みおちゃんの ふでばこから べつの みどりを とる** → みおちゃんは かなしそう。「かってに ふでばこを あけられて、びっくりしたんだって」→ こんちゃん「ひとの ものは、きいてから、だね」
- **C みどりを つかわずに かく** → くふうしたね!でも ほんとうは みどりが よかったんだよね。きもちは がまんしなくて いいんだよ
- **きもちクイズ**:(Aのあとの みおちゃんの かお)→ こたえ:うれしい。しあげの ふくしゅうなので ヒントなしで挑戦 → まちがえたら ヒント表示
- **設計メモ**:週の最終日。「5回目」を明示して1週間の積み上げを本人に見せる。金曜クリアで週間スタンプ完成の演出

## 3. 挿絵の画像生成プロンプト(Gemini量産用)

毎回**基準画像を添付**して、テンプレの【今回の場面】だけ差し替える。

**テンプレ**
```
添付画像の2人を、まったく同じキャラクターデザイン・輪郭線・配色・白背景で描いてください。

服装(全身が写る場合もこのとおりに固定):
- 男の子:くすんだ青の長袖、ベージュのズボン、青いくつ
- 女の子:くすんだ紫の長袖、ベージュのスカート、白いくつした、紫のくつ、おだんごの根元に紫の髪ゴム

添付画像に写っている小物(のりなど)は、下の場面指定にない限り描かないでください。

【今回の場面】(下記を挿入)

文字は入れない。横長16:9。
```

**量産の教訓(実運用で判明した事故と対策)**
- 基準画像の小物は場面が変わっても遺伝する → テンプレに「場面指定にない小物は描かない」を常設
- 全身が初めて写る場面で下半身の服装がランダムに決まる → 服装指定をテンプレに固定(上記)
- 感情を単語で書くと画像内に文字が湧く → 表情は「眉・口の形」で指定
- 髪ゴム・塗り残しなど細部は生成のたびに欠落しうる → 納品前チェックリスト:①髪ゴム(紫)②服の塗りむら③くつの左右の色④輪郭線の途切れ⑤余計なキラキラ・小物。細部修正を頼むときは「変えるな」の範囲も明示する(全体描き直しによる二次崩れ防止)

**「かして」週の場面指定文(問題画面用)**

| 日 | 【今回の場面】に入れる文 |
|----|--------------------------|
| 月 | 済(基準画像がそのまま使える) |
| 火 | 公園で、女の子がブランコに楽しそうに乗っている。男の子は少し離れて立ち、乗りたそうにためらいながら見ている。ブランコの支柱は最小限のシンプルな形で描く |
| 水 | 女の子がソファに座って携帯ゲーム機で遊んでいる。男の子はその隣で、やりたそうに手を少し伸ばしかけてためらっている |
| 木 | 図書室で、女の子が机で大きな動物図鑑を開いて読んでいる。男の子は本棚の横から、その図鑑を読みたそうに見ている。背景に本棚を1つだけ薄く描く |
| 金 | 教室の机で、女の子が緑の色えんぴつで絵を描いている。男の子は自分の絵を前に、緑の色えんぴつを借りたそうに女の子を見ている |

**結果画面用**は各日2枚(A成功:2人が笑顔で物の受け渡し/B結果:女の子が「びっくり」「おこってる」「かなしい」のいずれかの表情)。表情指定は「眉・口の形」で書くこと(感情の単語だけ書くと画像内に文字が湧く事故があるため)。例:「女の子は眉を下げ、口を小さくへの字にして悲しそうな表情」

                "title": "「かして」が いえたね!",
                "body": "みおちゃんは にっこりして かしてくれたよ。",
                "stamp": true

            { "type": "B", "label": "だまって つかう", "result": { "...": "..." } },
